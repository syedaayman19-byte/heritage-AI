Heritage AI is a modern web application designed to help people explore and preserve cultural heritage using artificial intelligence. 
The platform offers intelligent insights and detailed historical analysis, making cultural history accessible and engaging.
It features a clean, professional, and responsive user interface that works beautifully across all devices. The entire project is built using a highly modern tech stack including React, TypeScript, Vite, and Shadcn UI.
The application provides a structured platform to explore various aspects of Indian culture, including heritage sites, food, and art.
It includes a search functionality to help users quickly find relevant information.
The website is fully responsive and works across different devices. It also integrates the Wikipedia API to fetch real-time and reliable content dynamically, ensuring updated and informative results.
